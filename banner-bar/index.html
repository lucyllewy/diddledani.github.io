Banner bar web component

The above white area and the shadow beneath are an example of the <banner-bar> web component

Usage

<html> <head> <script src="path/to/platform.js"></script> <link rel="import" href="path/to/banner.hmtl" /> </head> <body> <banner-bar align="right" bgcolor="#fff" height="50px" maxWidth="960px"> <img src="example_logo.jpg" alt="example logo" /> </banner-bar> </body> </html>
align
Text alignment: left, center or right.
bgcolor
Background colour of the banner bar
height
height of the image which also specifies the height of the banner bar. If the image is smaller than this height it will be scaled.
maxWidth
The banner bar is responsive. This attribute specifies a "page width" whereby the image will right-aligned in a box the width specified here which will be centered in the page. Use "100%" to not use this feature.
